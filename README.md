# Scripting
En aquest repositori es treballaran scripts de cara al mòdul 0374 de 2n ASIX.

Es seguiran les idees i la teoria presentada a classe i es realitzaran pràctiques amb ajuda de la IA per idear els enunciats.
Seguint les pràctiques realitzades s'anira llistant l'enunciat de cada script en aquest fitxer per a tenir una primera idea abans d'endinsar-se en el codi.

## Repte 1 — Salutació personalitzada
Crea un script que rebi un nom com a argument i mostri per pantalla Hola, [nom]!. Si no es passa cap argument, ha de mostrar un missatge d'error explicant com usar l'script.

## Repte 2 — Calculadora bàsica
Crea un script que rebi tres arguments: un número, una operació (+, -, *, /) i un altre número, i mostri el resultat. Per exemple: ./calc.sh 10 + 5 ha de mostrar 15.

## Repte 3 — Comptador d'arguments
Crea un script que rebi qualsevol nombre d'arguments i mostri cada un d'ells numerats. Per exemple, si passes poma taronja pera, ha de mostrar:
```console
1: poma
2: taronja
3: pera
```

## Repte 4 — Processador d'opcions amb shift
Crea un script que processi les opcions -n (nom) i -e (edat) en qualsevol ordre. Per exemple: ./perfil.sh -n Anna -e 25 o ./perfil.sh -e 25 -n Anna han de mostrar el mateix resultat:
```console
Nom: Anna, Edat: 25.
```
